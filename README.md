# Simple Gallery

Projeto pessoal consumindo uma api aberta (https://picsum.photos/)<br/>
Possui testes

## Ferramentas

- React
- TypeScript
- styled-components
- axios
- react-query
- testing-library
- vite
- vitest
- msw

## Como usar

- Clone o projeto
- No terminal digite `npm i`
- Para executar o projeto digite `npm run dev`
- Para executar os testes digite `npm run test`
